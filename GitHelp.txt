Команды для регистрации ваших данных
1) git config --global user.name muralievv
2) git config --global user.email nmuraliev@gmail.com

Команды только для одного проекта
3) git init     Инициализаци Гита
4) git remote and origin link      Добавление репозитория

5) git status   Статус репозитория

6) git add .      Добавление файлов в буфер обмена
7) git commit -m "message"  Сохранение всего из буфера обмена

8) git push origin master    Отправка изменений в репозитории
9) git branch   Посмотреть все ветки
10) git branch new_branch   Создать новую ветку
11) git checkout new_branch  Переключаться на новую ветку
    git switch new branch  Переключиться на новую ветку
12) git restore .      Вернуть файлы в исходное состояние
13 git reset .        Вернуть файлы в исходное состояние после git add
14) git reset --mixed Head~1  Удалить последний коммит и вернуть файлы в исходное состояние
